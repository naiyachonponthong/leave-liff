ชื่อชื่อผู้ใช้บทบาทอีเมล สถานะจัดการ

คู่มือการใช้งาน

เลือกหัวข้อที่ต้องการ

1
ตั้งค่า LINE Official Account
  • เข้า LINE Developers Console → สร้าง Provider & Channel (Messaging API)
  • คัดลอก Channel Access Token และ Channel Secret
  • เปิด Webhooks → ใส่ URL ของ GAS Web App ที่ Deploy แล้ว
  • ปิด Auto-reply และ Greeting message
2
ตั้งค่า LIFF App
  • LINE Developers Console → Channel → แท็บ LIFF → Add
  • Size: Full · Endpoint URL: https://naiyachonponthong.github.io/leave-liff/
  • Scope: profile เท่านั้น
  • คัดลอก LIFF ID แล้วนำไปใส่ใน ตั้งค่า → LIFF ID
3
ตั้งค่าระบบหลังบ้าน
  • ไปที่เมนู ตั้งค่า → ใส่ Token, Secret, LIFF ID
  • เลือกระดับการอนุมัติ, วันทำงาน, เวลาทำงาน
  • กด บันทึกการตั้งค่า
  • กด ตั้งค่า Rich Menu เพื่อสร้างเมนูด้านล่าง LINE
4
เพิ่มพนักงาน & กำหนดบทบาท
  • ไปที่ พนักงาน → เพิ่มข้อมูลพนักงานแต่ละคน กำหนด Role
  • พนักงานเปิด LINE → กด Rich Menu → กรอกรหัสพนักงาน + เบอร์โทร
  • ระบบจะผูก LINE userId อัตโนมัติ
✅ พร้อมใช้งานแล้ว — ระบบมีปุ่มอนุมัติ/ไม่อนุมัติใน LINE อยู่แล้ว ไม่ต้องตั้งค่าเพิ่ม
1
พนักงานยื่นใบลา

ผ่าน LINE LIFF → กด ยื่นใบลา

2
ผู้อนุมัติได้รับ LINE Flex Message

มีปุ่ม อนุมัติ และ ไม่อนุมัติ กดได้เลยในแชท LINE

3
ระบบตอบกลับและแจ้งพนักงาน

ผลอนุมัติส่งกลับใน LINE แชท และ push แจ้งพนักงานทันที

ข้อกำหนด: ผู้อนุมัติต้องเป็น Friend กับ LINE OA และต้องมี LINE userId บันทึกในระบบ (ผูกบัญชีก่อน)
◆
สร้างใบลาแทนพนักงาน

เมนู ใบลา → + สร้างใบลา → เลือกพนักงาน ระบบคำนวณวันและหักยอดอัตโนมัติ

◆
จัดการประเภทการลา

เมนู ข้อมูลหลัก → แท็บ ประเภทการลา — กำหนดโควตา, สี, ต้องแนบเอกสาร, ลาครึ่งวัน, โควตาตามอายุงาน

◆
ยกยอดวันลาข้ามปี

เมนู ตั้งค่า → กด ยกยอดวันลาปีที่แล้ว ทำต้นปีใหม่ ระบบนำวันคงเหลือไปบวกปีถัดไปอัตโนมัติ

◆
อนุมัติหลายรายการพร้อมกัน (Bulk)

เมนู คิวอนุมัติ → ติ๊ก checkbox → กด อนุมัติทั้งหมด หรือ ไม่อนุมัติทั้งหมด

◆
อนุมัติผ่าน LINE (แนะนำ)
  • รับ Flex Message ใน LINE เมื่อมีใบลาใหม่
  • กดปุ่ม อนุมัติ หรือ ไม่อนุมัติ ในแชทได้เลย
  • ระบบตอบกลับผลทันที ไม่ต้องเปิดหน้าเว็บ
◆
อนุมัติผ่านหน้าเว็บ

เมนู คิวอนุมัติ → กด อนุมัติ หรือ ปฏิเสธ → ใส่ความเห็น → ยืนยัน ระบบแจ้งพนักงานทาง LINE

1
เริ่มใช้งานครั้งแรก
  • เพิ่มเพื่อน LINE Official Account ของบริษัท
  • กด Rich Menu ด้านล่างแชท → หน้า "เชื่อมต่อบัญชี"
  • กรอก รหัสพนักงาน + เบอร์โทรศัพท์ → กด เชื่อมต่อ
2
ยื่นใบลา

แท็บ ยื่นลา → เลือกประเภท → เลือกวัน → เหตุผล → แนบเอกสาร → กด ยื่นใบลา

3
ดูประวัติ / ยกเลิก

แท็บ ประวัติ → กรองด้วย chip สถานะ → กดที่ card เพื่อดูรายละเอียด → ยกเลิกได้หากยังรออนุมัติ

4
วันคงเหลือ & วันหยุด

แท็บ คงเหลือ → ดูโควตาแต่ละประเภทพร้อม progress bar · แท็บ วันหยุด → รายการวันหยุดปีนี้
